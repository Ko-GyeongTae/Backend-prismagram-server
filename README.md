- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] Edit my profile
- [ ] Log in
- [ ] Upload a photos
- [ ] Edit the photo (Delete)
- [ ] See the feed
